Instructions for running AlphaFold heme_insert_script

>Open AlphaFold2 c-type heme protein model structure in PyMOL.
>Go to file
>Go to Run Script...
>Open C_heme_insert_script_V6.pml

This should run the script and save a copy of the output structure in your current working directory.
Please note the script will not perform as well for non-canonical c-type heme attachment motifs, or when the AlphaFold model supplied is itself of poor quality.

If you experience issues, please contact Benjamin.Nash5@outlook.com
